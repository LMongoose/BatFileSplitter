LINUX:
split -l 100 targetfile
By default split makes output files xaa, xab, and so on.

split -d -l 100 targetfile PREFIX
This command will make files PREFIX01, PREFIX02, and so on. The -d option is a GNU extension, so it isn't supported on all systems.

WINDOWS:
If you have "git bash", use the linux split
if don't have it, download it from: https://git-scm.com/
